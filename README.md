# express-blog
Demo project for blog posting and approval from the backend, built with Express JS, featuring a super admin and a Category manager

## Table of Contents

## Features

- Signup/Login 
- Creating blogs from user side
- Topic manager with approval / rejection 
- Super admin for creating topics and assigning topic managers

## Installation

1. Clone the repository: `git clone https://github.com/AkhilTreblle/express-blog`
2. Navigate to the project directory: `cd express-blog`
3. Install dependencies: `npm install`

## Usage

1. Run the development server: `node index`
2. Visit `http://localhost:3030` in your web browser.
